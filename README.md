# Customer Conversion Analytics

## Objective
Analyze trial-to-paid conversion behavior using cohort analysis.

## Dataset
- 10,000 customers
- Customer demographics
- Trial lapse dates
- Subscription dates
- Subscription prices

## Techniques Used
- Cohort Analysis
- 7-Day Conversion Rate
- 14-Day Conversion Rate
- Country Segmentation
- Rolling Averages
- Exponential Weighted Moving Averages (EWM)

## Tools
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Conversion varies across countries.
- 14-day conversion is consistently higher than 7-day conversion.
- EWM helps identify long-term trends by reducing daily volatility.
